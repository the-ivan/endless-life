# Endless Life

![Endless Life Demo](assets/endless-life-demo.gif)

Conway's Game of Life for the Nothing Phone (3) Glyph Matrix.

## Features

- Continuous Game of Life on the 25×25 Glyph Matrix
- Time-seeded patterns with stable/extinct detection and ending fade
- Eight starting reveal animations (configurable)
- Long-press Glyph button to force a new life
- Resumes mid-simulation after toy bind/unbind (e.g. clock carousel)

## Install from GitHub Releases

1. Open [github.com/the-ivan/endless-life/releases](https://github.com/the-ivan/endless-life/releases) on your phone (or download the APK on a PC and transfer it).
2. Under the latest release, download **endless-life-1.0.0.apk** (version in the filename matches the release tag).
3. Optional — verify the APK on [Koodous](https://koodous.com/apks/e7a6c8468b8048f5996f5638816166102c024d45056fe8a8dc9261b9c0536bf9) before installing.
4. Open the downloaded file and install. If Android blocks it, allow installs from your browser or Files app when prompted.
5. Launch **Endless Life** and tap **Open Glyph Toys Manager**, then enable **Endless Life**.
6. Optional — Always-on: in Glyph Toys Manager, set **Endless Life** as your Always-on Glyph Toy (Flip to Glyph). In the app, tap **Open App Settings** and set Battery background usage to **Unrestricted**.

## Build from Source

1. Download [Glyph Matrix SDK 2.0](https://github.com/Nothing-Developer-Programme/GlyphMatrix-Developer-Kit) into `app/libs/`.
2. For release builds, create `keystore.properties` at the repo root with `storeFile`, `storePassword`, `keyAlias`, and `keyPassword` (local only — not committed).
3. `./gradlew assembleRelease`
4. `adb install app/build/outputs/apk/release/endless-life-1.0.0.apk`

## Settings

Animation selection, simulation speed, initial density, and resume toggle. Changes apply on the next life or after re-selecting the toy as Always-on.  

For uninterrupted Always-on play on battery, go to App Settings for Endless Life and set battery background usage to Unrestricted.
